gab.operations.add_ssh_config(hostname, username, identity_file)[source]¶

Add a line to the ssh config file (found in ~/.ssh/config).

E.g. the configuration for github:

add_ssh_config('github.com', 'git', '~/.ssh/my_gh_key)

will result in:

Host github.com
    User git
    Hostname github.com
    IdentityFile ~/.ssh/my_gh_key
Parameters:
  • str (identity_file) – the external host
  • str – the username you need to use when you connect to that host, specify * if it’s for all usernames
  • str – the location to the private key file
